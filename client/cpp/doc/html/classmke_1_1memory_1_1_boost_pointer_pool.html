Detailed Description

template<class T, class Compare = std::less<T>>
class mke::memory::BoostPointerPool< T, Compare >

Implements a thread-safe memory pool of pointers. Once a pointer is added into the memory pool via the add method, the PointerPool takes ownership of the pointer and deallocates it in its destructor or via the clear method.
